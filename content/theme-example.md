# Theme Switching

MDNet now supports both **Solarized Dark** (default) and **Solarized Light** themes.

## How to Switch Themes

Click the **☀/☾** button in the top-right corner to toggle between light and dark modes.

- **☀ (Sun icon)** = Currently in dark mode, click to switch to light
- **☾ (Moon icon)** = Currently in light mode, click to switch to dark

## Theme Persistence

Your theme preference is saved to browser localStorage and will persist across:
- Page navigation
- Browser sessions
- Page refreshes

## Color Schemes

### Solarized Dark (Default)
- Background: `#002b36` (dark blue-gray)
- Text: `#93a1a1` (light gray)
- Links: `#268bd2` (blue)
- Code: `#2aa198` (cyan)

### Solarized Light
- Background: `#fdf6e3` (cream)
- Text: `#657b83` (dark gray)
- Links: `#268bd2` (blue)
- Code: `#2aa198` (cyan)

## Features

Both themes use the complete Solarized color palette:
- Consistent color values across themes
- Smooth transitions when switching
- Optimized for readability
- Carefully designed contrast ratios

## Try It Now!

Click the theme toggle button now to see this page in the other theme. Notice how:
- All colors smoothly transition
- Syntax highlighting adapts
