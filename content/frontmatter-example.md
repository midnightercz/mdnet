# Front Matter Example

This page demonstrates the front matter support in MDNet.

## What is Front Matter?

Front matter is metadata about the page, written in YAML format at the top of the markdown file. It's enclosed between `---` delimiters.

## Features

- Automatically parsed from markdown files
- Displayed in a collapsible table below the title
- Collapsed by default to keep the page clean
- Supports various data types: strings, numbers, arrays, objects

## Example Front Matter

```yaml
---
author: John Doe
date: 2024-04-30
tags: [example, frontmatter, demo]
category: Tutorial
status: published
---
```
