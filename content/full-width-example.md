# Full Width Example

This page uses **100%** width to maximize the available screen space.

## Configuration

```yaml
md-page-width: 100%
```

## Use Cases

Full width layouts are ideal for:

1. **Data Tables** - Wide tables with many columns
2. **Dashboards** - Multiple widgets side-by-side
3. **Code Examples** - Long code lines without wrapping
4. **Image Galleries** - Displaying images at full resolution

## Example: Wide Code Block

```javascript
const exampleFunction = (param1, param2, param3, param4, param5) => {
  return `This is a very long line of code that benefits from full width display without horizontal scrolling or wrapping to maintain readability`;
};
```
